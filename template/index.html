About Rainy Brain

Rainfall Predictor is a web app which has a Machine Learning model running at the back. The purpose of developing this app is to predict whether it will rain the next day or not. This model is based on the Rain Prediction in Nepal dataset. The codes for this project can be checked in github repo.
